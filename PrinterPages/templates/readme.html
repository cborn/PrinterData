Short Description of Project

Currently the site has 3 pages: a homepage, an advanced search page, and a search results page. The homepage allows the user to search for anything. Anything in the database that matches the query is returned in order of popularity (aka number of downloads). This same search principle is applied to the small search bar in the upper right corner of the two other pages. The advanced search page is reachable from the search results page and the homepage. It allows the user to specify which section they would like to search in (author, title, or genre). The Project Gutenberg logo persists through all pages and is click-able.

List of Features

  • Search for anything and get results back sorted by popularity.
  • Narrow down the search by querying specific categories.
  • Displays helpful information about free eBooks.
  • Link directly to the actual eBooks.

Data Set Citation

  • Name: "The Complete Project Gutenberg Catalog"
  • Organization: Project Gutenberg
  • Link to the data set
  • Licensing information
